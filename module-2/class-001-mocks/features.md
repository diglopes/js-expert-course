# Features

APP to convert from **CSV** to **JSON** format respecting the following bussiness rules:

- **R1:** The file must have id, name, profession and age fields
- **R2:** The file must not be empty
- **R3:** The file should't exceed 4 lines (headers included)
  
> ⚠️ **Important lesson:** you should always write down the needs of your system before even touch the code. First of all, have a clear idea of what you should do.
